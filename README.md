# 🎵 Terminal Music Player (Windows)

---

## ✨ 功能特性

* **双模式切换**：既有漂亮的终端交互 UI，也支持完全通过命令行参数控制（便于通过龙虾运行）。
* **三种播放模式**：顺序播放、随机播放、单曲循环。
* **后台自动续播**：智能监控播放进程，曲终自动下一首。
* **断点记忆**：自动保存上次播放的曲目、进度、音量及模式。
* **无缝集成**：支持通过其它脚本（如 Python 自动化管线）进行远程控制。

---

## 🛠️ 重要！！ 环境准备

本播放器依赖 **Python 3.10+** 以及 **FFmpeg (ffplay)**。

### 1. 安装 Python
确保你的系统中已安装 Python，并建议在虚拟环境（如 Conda）中运行。

### 2. 下载并安装 FFmpeg (必选)
播放器使用 FFmpeg 套件中的 `ffplay` 作为解码引擎。请按以下步骤安装：

1.  **下载**：访问 [Gyan.dev](https://www.gyan.dev/ffmpeg/builds/)，下载 `ffmpeg-release-essentials.zip`。
2.  **解压**：将下载的压缩包解压到一个固定位置（例如 `C:\ffmpeg`）。
3.  **配置环境变量**：
    * 右键“此电脑” -> “属性” -> “高级系统设置” -> “环境变量”。
    * 在“系统变量”中找到 `Path`，点击“编辑”。
    * 点击“新建”，输入 FFmpeg `bin` 文件夹的路径（例如 `C:\ffmpeg\bin`）。
    * 点击“确定”保存。
4.  **验证**：打开终端输入 `ffplay -version`，如果显示版本信息，则安装成功。
